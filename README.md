# CODSOFT Machine Learning Internship Tasks

This repository contains the projects completed during the Machine Learning internship at **CodSoft**. The internship involved building various predictive models using real-world datasets, focusing on natural language processing, financial security, and customer analytics.

## 🚀 Projects Overview

I have completed four distinct tasks using different machine learning algorithms and techniques. Below is a summary of each project:

### 1. Movie Genre Classification
- **Objective:** Predict the genre of a movie based on its plot summary.
- **Algorithms:** TF-IDF Vectorization & Multinomial Naive Bayes.
- **Outcome:** Successfully classified movies into multiple categories with an accuracy of **52.18%**.

### 2. Credit Card Fraud Detection
- **Objective:** Build a model to identify fraudulent credit card transactions.
- **Algorithms:** Random Forest Classifier.
- **Outcome:** Achieved a high accuracy of **99.75%**, effectively distinguishing between legitimate and fraudulent activities even in highly imbalanced data.

### 3. Customer Churn Prediction
- **Objective:** Predict whether a customer will churn (leave) a subscription-based service.
- **Algorithms:** Gradient Boosting Classifier.
- **Outcome:** Achieved an accuracy of **86.60%** using features like customer demographics and usage behavior.

### 4. Spam SMS Detection
- **Objective:** Classify SMS messages as either spam or legitimate (ham).
- **Algorithms:** TF-IDF Vectorization & Support Vector Machine (SVM).
- **Outcome:** Highly accurate detection with **97.49%** accuracy, ensuring reliable filtering of spam messages.
